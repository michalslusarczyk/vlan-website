Przewodnik techniczny

Dokument roboczy: co, gdzie i jak jest zaimplementowane w projekcie.

HTML

Struktura stron

Core
  • index.html — strona główna (noc).
  • day.html — strona główna (dzień).
  • panel.html — panel po logowaniu Google.
  • subcard-templates.html — biblioteka podkart.

JS

Logika aplikacji

Flow
  • script.js — sekcje dynamiczne, popupy, karty, sklep, kontakt.
  • auth.js — logowanie Google i ochrona widoków panelowych.
  • language.js — przełączanie języka PL/EN.
  • auth-config.js — konfiguracja Google Client ID.

CSS

Style bazowe i komponentowe

  • style.css — design system, sekcje, karty, podkarty, panel.
  • style-day.css — nadpisania dla trybu jasnego.
  • Podkarty: subcard + warianty base/accent/kpi/compact/horizontal.

ROZWÓJ

Jak dodawać nowe moduły

Guide
  1. Wybierz model podkarty z subcard-templates.html.
  2. Dodaj markup sekcji w odpowiednim pliku HTML.
  3. W razie potrzeby dopisz logikę do script.js.
  4. Sprawdź tryb nocny i dzienny przed publikacją.

Notatki wdrożeniowe
